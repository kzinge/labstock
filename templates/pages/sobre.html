O labstock tem como objetivo desenvolver um sistema de gerenciamento de estoque virtual para o laboratório de química do IFRN Campus Caicó, visando otimizar a gestão de recursos e melhorar a organização dos materiais. Para isso, foi feito um levantamento com usuários, como professores e técnicos, para identificar as principais dificuldades enfrentadas na gestão do estoque. Com isso, a metodologia incluiu a definição de requisitos funcionais e não funcionais, além da utilização de ferramentas como Figma para design da interface, HTML e CSS para estrutura, JavaScript para interatividade, e Python com Flask para o backend, utilizando MySQL como banco de dados. Os resultados indicam que a implementação do sistema permitirá um controle eficiente de materiais, reagentes e equipamentos, assegurando a disponibilidade dos recursos e facilitando o trabalho dos usuários. A ferramenta não só otimizará a organização do estoque, mas também possibilitará a publicação e compartilhamento de dados, promovendo transparência e colaboração. Portanto, concluímos que a criação do almoxarifado virtual não apenas moderniza a gestão de laboratórios, mas também promove um ambiente de trabalho mais integrado. Além disso, a proteção legal das inovações é essencial para garantir que possam ser reconhecidas e exploradas de forma segura, contribuindo para o avanço tecnológico na área.
